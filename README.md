# banner_jeong

## descript
여러가지 기술을 사용하는 커맨드의 모음집 

## install
```bash
$ pip install banner_jeong==0.3.0
```

## Usage

### 특정 텍스트 배너 형태로 표출
```bash
$ show-banner
```

### 로또 번호 생성
```bash
$ lotto
```

### 특정 그림 형태로 표출
```bash
$ pic
```
